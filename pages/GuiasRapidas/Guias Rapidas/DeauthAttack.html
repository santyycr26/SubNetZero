Death Attack - Como realizar un ataque de Deauth Attack para auditorias Wifi

Muchas veces a la hora de realizar auditorías es esencial abarcar todos los campos y una de ellas es la seguridad WIfi, en esta guía se explota
una vulnerabilidad de WPA-2 que consiste en desautenticacion de la red Wifi y capturar el handshake para poder realizar un ataque de diccionario

Aircrack-ng: nuestra herramienta amiga

¿Cómo instalarlo?
En linux debian: sudo apt install aircrack-ng
En windows: Directamente desde la página web

Primero antes que nada, se debe tener una antena Wifi compatible con modo monitor(en este lab se usará un Intel Killer Wifi 6 ax1650
